This app is a EMT-B simulator training web app, with a sleek minimilistic appearance. This web app should automatically create a simulation when the website is loading and set up vitals that match the situation. The simulator is built around a Human object that can be onjured and sick in a variety of ways that mimic real life pathology. The user has the capabilities outlined in "Treatment Protocols MASS.pdf". The user can choose treatments in a dropdown, see vitals with a button press, and see the clock time. There should also be a transport button starting time to hospital. 

The initial version of this program can do anaphylaxis and asthma managment and has the following vital signs, heart rate, respitory rate, SBP, DBP, lung sounds, spo2, temp, airway patency, skin signs, capillary refill time, and GI condition.
